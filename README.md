## Installation

```bash
$ git clone https://github.com/wavygoten/ig-hashtag-comment-bot.git
$ cd ig-hashtag-comment-bot
$ npm i
$ node bot.js
```

or

```bash
$ git clone https://github.com/wavygoten/ig-hashtag-comment-bot.git
$ cd ig-hashtag-comment-bot
$ yarn
$ node bot.js
```
